# The NEAX Platform Repository

This repo contains the source code for **Nigeria Energy Access Explorer (NEAX)**.

NEAX is an open, Nigeria-focused geospatial planning project for identifying and prioritizing places where energy access can be expanded. The goal is to combine demand and supply signals into clear, transparent rankings that planners, NGOs, and researchers can use for real-world decision-making.

Right now, the project is in active MVP development. The current focus is building a reliable data pipeline and scoring workflow first, then layering in richer datasets and map-based exploration.

I started this project to sharpen my engineering skills across data pipelines, geospatial workflows, and full-stack product development, while building something practical and useful in the real world. 

## Project Areas

- `data-pipeline/` - ingestion, transformation, scoring, and artifact generation
- `apps/web/` - frontend application (early-stage)
- `docs/planning/` - problem framing, PRD, ADRs, and technical decisions
- `docs/references/` - methodology notes and reference materials
